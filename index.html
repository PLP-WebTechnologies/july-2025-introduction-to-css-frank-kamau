Overview

This page uses an external stylesheet (styles.css) to style headings, paragraphs, navigation and layout sections. The CSS demonstrates the box model by using padding, border, and margin on elements and visualizing boxes with background colors.

Box Model Examples

Below are three boxes that intentionally show different uses of padding, border and margin. Notice how the space inside the box (padding), the border, and the space outside the box (margin) change the layout.

Margin

External spacing from surrounding elements.

.margin-demo { margin: 30px; }

Padding

Internal spacing between content and border.

.padding-demo { padding: 30px; }

Border

The visible edge of a box; also contributes to total size.

.border-demo { border: 6px solid rgba(11,99,255,0.25); }

Tip: Open your browser devtools and inspect any of these elements. The devtools "box model" panel visualizes margin, border, padding and content area for the selected element.
